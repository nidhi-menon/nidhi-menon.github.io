API Log Compression

Autodesk — reducing storage costs on high-volume API telemetry

About

Designed and shipped a log compression pipeline for high-throughput API telemetry, reducing storage costs by ~60% while preserving full fidelity for anomaly detection and audit queries. The system included a sampling strategy specifically tuned to preserve tail-latency signal that would otherwise be lost under naive compression.

Details

  • ~60% reduction in storage costs with no loss of audit or anomaly detection fidelity
  • Tail-latency-preserving sampling strategy to retain high-percentile signal
  • Streaming pipeline via Kafka with Parquet-based cold storage on S3
  • Query layer via Athena enabling ad-hoc analysis without full log reconstruction

Technologies

Python Apache Kafka S3 Parquet Athena
