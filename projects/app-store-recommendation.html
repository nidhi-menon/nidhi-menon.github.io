App Store Recommendation System

Autodesk App Store — personalized discovery at scale

About

Designed and shipped a recommendation system for the Autodesk App Store that surfaces relevant third-party integrations to CAD/BIM users. Combined collaborative filtering with content-based signals from user behavior and app metadata to improve discoverability across a catalog of 1,500+ apps. The system served 708K daily active users at peak.

Details

  • Led end-to-end design and delivery across modeling, infrastructure, and API integration
  • Combined collaborative filtering with content-based signals for cold-start handling
  • Built on Wide & Deep architecture to balance memorization and generalization
  • Served real-time recommendations via low-latency API backed by DynamoDB

Technologies
